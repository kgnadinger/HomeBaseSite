# Screenshots Directory

Place your app screenshots here. The Screens component expects the following files:

- `onboarding.png` - Onboarding screen
- `parent-dashboard.png` - Parent dashboard
- `create-chore.png` - Create chore screen
- `child-dashboard.png` - Child dashboard

The component will automatically display these screenshots in a responsive grid layout. If a screenshot is missing, a placeholder will be shown.
